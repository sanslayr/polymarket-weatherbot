# RJTT / Tokyo

手工站点背景备注，供新站点接入阶段使用；历史 METAR 画像后续再补全。

## 站点定位

- 站点：`RJTT` / Tokyo Haneda
- 类型：东京湾填海机场，不等同于东京市中心热岛
- 固定标签：`湾岸填海机场`
- 关键方位：东南侧临水，主城位于北到西北侧

## 背景判断

- RJTT 的主导逻辑是湾风和城市边界层切换，不是单纯“东京很热所以机场也会一直热”。
- 偏西到偏北陆地背景更容易维持偏暖上限；东到东南湾风切入后，湿层和云量常会很快压平尾段升温。
- 这个站点的日较差通常比东京内陆城区更小，午后封顶也往往更早，需要特别盯风向切换时点。

## 监控顺序

1. 先看是否仍维持偏西/偏北陆地背景。
2. 再看东南侧湾风和海层湿空气何时切入。
3. 最后结合低云、露点和阵雨残留判断峰值窗是否被截断。
