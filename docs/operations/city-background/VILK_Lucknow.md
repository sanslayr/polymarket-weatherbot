# VILK (Lucknow) 城市背景调研（用于 Polymarket 最高温）

- 站点：**VILK / Lucknow (Amausi)**
- 坐标：26.7606N, 80.8893E
- 地形标签：内陆平原（恒河平原中部）
- 更新时间：2026-03-04 UTC

---

## 1) 结论摘要（交易/实况修正向）

1. **无对流触发日**：温度日变化相对稳定，最高温通常由“晨间低能见现象散除时点 + 午前混合层发展”决定。  
2. **恒河平原背景下，晨间 BR/HZ 很常见**，会显著影响升温启动时间。  
3. **对流季（前季风/季风）要切换范式**：CAPE/CIN/LFC 与触发温度的权重应明显上调，防“峰值窗被对流截断”。

## 3) 恒河平原（Lucknow）对最高温的关键机制

### A. 非对流主导日（多数冬春日）
- 夜晨易出现 BR/HZ/FG，早段短波被削弱。
- 若 10–12L 前后能见度改善、T-Td 扩张，升温会进入“追赶段”。
- 峰值常落在 **13–15L**，存在后移特征。

### B. 对流主导风险日（前季风/季风）
- 典型风险期：
  - **前季风：4–6月**（午后触发敏感）
  - **季风主雨季：6–9月**（云雨对 Tmax 截断明显）
  - **退季风过渡：10月初**（仍有残余触发）
- 关键风险：峰值窗前后若触发降水/雷暴，Tmax 上沿会被快速压制。

## 4) 实战监测模板（VILK）

### 4.1 平稳日（无明显对流）
优先监测：
1. 日出后 2–4h 能见度变化（BR/HZ/FG 是否快速减弱）
2. `T-Td` 扩张速度（混合层有效性）
3. 风速是否从静风转弱混合风（2–6kt 到更可混合）

修正建议（经验向）：
- 若 09–12L 仍低能见 + 斜率钝化：下修上沿并后移峰值窗。
- 若午前快速转晴且斜率恢复：撤回部分下修。

### 4.2 对流风险日
优先监测（sounding/模式）：
- CAPE / CIN / LCL / LFC
- 触发温度（Convective Trigger Temperature）
- 0–3km 湿度与低层风切变

修正建议：
- 若触发条件接近达成且峰值窗重叠：削减冲高尾部概率，防“对流截断”。
