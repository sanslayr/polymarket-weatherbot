## 13) 旧版易误判场景

1. 按 downtown Chicago 的热岛路径外推，会高估 O'Hare 暖季峰值。
2. 忽略 frontal wind shift，只看同小时温度，会把后侧冷平流日误判成暂时偏冷。
3. 若不单独处理雨后重置日，暖季上限会被系统性高估。
4. 晨间低能见/湿层较重时，启动偏慢更常见，不宜直接套 clean-ramp 日模板。

## 14) 实况优先观察顺序

1. 先看风是否留在 S/SW 暖区，再看云/降水残留，最后评估午后还能否保持升温斜率。
2. 风向切换应视为 live pivot，必要时比同小时模式偏差更值得信任。
3. 有降水或地面湿润残留时，优先防高估；除非午前快速放晴且斜率恢复。
4. Jun 常见干混合上冲，晴空低云少的日子要防旧版保守低估。
