## 13) 旧版易误判场景

1. 把所有暖季日都按干热清空日外推，会高估阵雨残留日。
2. 把午前偏慢直接视为全天失败，也会漏掉开窗后再追高的情况。
3. 若在午后前段就提前封顶，最容易低估真正的晚峰或尾段冲高。
4. 若不单独处理雨后重置日，暖季上限会被系统性高估。
5. 晨间低能见/湿层较重时，启动偏慢更常见，不宜直接套 clean-ramp 日模板。

## 14) 实况优先观察顺序

1. 先排查早段是否有降水重置，再看云层残留和露点，最后看 SW/W 暖通道是否建立。
2. 接近窗口后段仍有斜率时，应继续保留上冲空间，避免过早封顶。
3. 露点和湿层厚度的解释优先级要提高，单看温度斜率容易误判。
4. 风向切换应视为 live pivot，必要时比同小时模式偏差更值得信任。
5. 有降水或地面湿润残留时，优先防高估；除非午前快速放晴且斜率恢复。
