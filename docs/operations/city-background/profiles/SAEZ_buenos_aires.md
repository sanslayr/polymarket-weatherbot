## 1) 结论摘要（历史 METAR + 站点背景）

1. 河口平原大日较差站，北到东北暖平流与南侧冷空气重置切换非常关键。
2. 冷季(6-8)偏暖风向 `NE(18.6°C) / N(18.3°C)`；偏冷风向 `W(14.0°C) / S(13.0°C)`；温差约 `5.0°C`；峰值时刻约 `16:00` vs `15:30`。
3. 晚峰占比高，若 15L 后仍保有斜率，不宜提前按内陆早峰模板封顶。
4. 风向切换敏感，主导风向组合为 E:23%, S:16%, NE:15%，转向常常比当前温度更早发出路径变化信号。
5. 日较差偏大，晴空干混合日的上冲空间明显高于站点日常印象。
6. 同月最热窗口通常出现在 Jan，对应月度 Tmax 中位约 31.1°C。

## 2) L2 核心识别框架

## 13) 旧版易误判场景

1. 忽略南风切入后的快速降档，会把暖区延续看过头。
2. 把 Buenos Aires 全部按海边平缓日变化处理，会低估内陆式大日较差。
3. 若在午后前段就提前封顶，最容易低估真正的晚峰或尾段冲高。
4. 晨间低能见/湿层较重时，启动偏慢更常见，不宜直接套 clean-ramp 日模板。

## 14) 实况优先观察顺序

1. 先看是否维持在 N/NE 暖区，再看冷空气/南风何时推进，最后看云量是否会截断 16L 后的尾段上冲。
2. 接近窗口后段仍有斜率时，应继续保留上冲空间，避免过早封顶。
3. 风向切换应视为 live pivot，必要时比同小时模式偏差更值得信任。
4. Dec 常见干混合上冲，晴空低云少的日子要防旧版保守低估。
