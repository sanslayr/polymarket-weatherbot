# Bug Reports Archive

本目录用于归档 `/look` 相关缺陷报告（bug report / bugfix 说明）Markdown。

## 存放规则
- 所有同类问题报告统一放在本目录，不再散落在 `docs/operations/` 根目录。
- 文件命名建议：`BUGFIX_<STATION_OR_SCOPE>_<SHORT_TOPIC>.md`。
- 内容建议包含：`Issue`、`Repro`、`Root Cause`、`Fix Strategy`、`Regression Cases`、`Expected Outcome`。
