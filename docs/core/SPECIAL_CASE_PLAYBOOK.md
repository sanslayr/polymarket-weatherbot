# /look 天气形势特殊情形手册（可扩展）

Last updated: 2026-03-02

> 本文件只归档**天气形势相关**规则（synoptic/云雨/风场/窗口行为）。
> 技术实现细节（市场拉取、F/C 解析、METAR 量化等）已分离到 `TECHNICAL_IMPLEMENTATION_NOTES.md`。

---

## 1) 使用范围
- 适用于 `/look` 的跨城市最高温判断。
- 目标是回答：
  1) 今天高点是否已定？
  2) 是否还有条件反超？
  3) 哪个变量最可能改写结论？

---

## 2) 核心天气形势规则

### A. 早段已出高点（early-day max）
**触发：** `phase=post` 且已有 `observed_max_temp_c`

**判定目标：** 不再泛化“窗口后已定”，而是直接判断“还能否反超前高”。

**流程要求（先后顺序）：**
1) 先构建“潜在二峰窗口”（post-focus secondary window）
2) 再围绕该窗口做环流/探空链接与解释
3) 最后做“反超门槛 + 可行性”输出

**程序约束（硬规则）：**
- `post` 阶段默认压制上沿尾部（skew 收敛）。
- 若当前 `BKN/OVC/VV` 或降水在场且斜率弱，则上沿执行保守 cap（守高优先）。
- 若潜在二峰模型峰值本身明显低于已观测前高（如低 ≥0.4°C），强制按“不可反超”口径处理。

**仅在以下条件同时满足时放宽：**
- 云层持续开窗（非 BKN/OVC/VV）
- 无降水干扰
- 温度斜率连续转强
- 距前高时间仍合理（非久后硬冲）

---

### B. 云层约束与开窗
**低云压制（BKN/OVC/VV 持续）**
- 上沿下压
- 尾部缩短
- 若同时有降水/弱斜率，则优先按“守高”处理

**开窗信号（减弱/开窗）**
- 允许小幅恢复上沿
- 但必须与温度斜率协同，不单凭云层判断

---

### C. 降水演变优先级
降水演变是一级方向变量：
- `new / intensify`：压温风险上升，上沿下修
- `steady`（且中等以上）：增温效率受抑
- `end + 低云仍在`：残余冷却延续，不可立即暖反弹

---

### D. 冷暖平流与落地可达性
**冷平流（尤其 far-window + 北向来流）**
- 需限制“快速反弹”预期，防止过暖偏差。

**暖平流**
- 只能作为“支撑因子”，不是自动抬升结论。
- 必须与云量/降水/斜率协同才放大上沿。

---

### E. 午后平台期（辐射边际下降）
在 `near/in window`：
- 若已出现平台且斜率偏弱，应抑制乐观上冲尾部。
- 仍可小幅上探，但不能假设持续加速。

**晴空圆弧顶（典型高估场景）补充：**
- 若 `CLR/CAVOK/FEW/SCT` 持续，且临窗阶段温度斜率已走平/减弱（含两步加速度转负），应触发“圆弧顶锁高”约束。
- 该约束应优先压制惯性上冲尾巴（尤其在已接近当日观测高点时）。
- 暖平流只允许小幅放宽，不应覆盖圆弧顶锁高结论。
- 引入 METAR 多层云量 + 天气现象的有效辐射因子 `R_eff`：
  - `R_eff` 低且不回升 → 上沿压制增强
  - `R_eff` 高且继续回升 → 允许末段小幅冲高，不做一刀切压顶

### F. 晨冷午暖偏差翻转（日振幅低估）
**典型信号：** 同一天内出现“清晨偏冷、白天转偏暖”的偏差翻转。

**常见机制：**
- 夜晨近地逆温/冷池更强 → 模式清晨偏暖（实况更低）
- 日照后湍混合加速/开窗更快 → 模式白天偏冷（实况更高）

**判读规则：**
- 不是随机噪声，优先判作“模式日变化振幅被压小（且可能相位偏慢）”。
- 在 `far/near_window` 阶段，若偏差由负转正且升温斜率强于模式，应上调 Tmax 上沿权重。
- 但仅在“云量约束弱 + 无降水压温 + 风场不转入冷向增强”时放宽；否则维持保守上沿。

**输出建议（一句话模板）：**
- `今日出现晨冷午暖偏差翻转，模式低估白天升温振幅；若中午前后斜率继续强于模式，最高温上沿仍可小幅上修。`

### G. 夜间增温（after-sunset reheat）
**触发场景：** 峰值窗后/日落后，需判断是否仍存在小幅回升（非白天辐射驱动）。

**夜间增温常见来源（组合判定）：**
- 暖平流落地（不是“提到暖平流”就算，需实况偏差/趋势配合）
- 边界层混合作用增强（风速上跳并达到可混合阈值）
- 云被回补（SCT/BKN/OVC 回补，抑制地面长波冷却）
- 露点回升与气压走低（暖湿平流/系统靠近的辅助证据）

**约束原则：**
- 夜间增温仅放宽为“**小幅回升可能**”，不等同于白天强冲高。
- 若无明显暖平流，且仅有晴空/弱风横盘，默认继续按“上冲概率低”处理。
- 降水因子要单独扣分：
  - `new/intensify`（含小雨）按硬抑制处理；
  - `end/weaken + BKN/OVC` 视作残余冷却，降低夜间回升概率；
  - 中等及以上降水在场时，夜间增温放宽应失效。

## 3) 窗口后（post）输出导向
窗口后优先输出“反超前高门槛”而不是泛化描述：
1. 还差多少°C 才能反超
2. 现有条件支持/不支持（云、雨、斜率、风场）
3. 最快改判触发（下一报斜率 + 云层变化）
